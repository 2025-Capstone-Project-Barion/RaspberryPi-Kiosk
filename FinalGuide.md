🛠️ 전시회 현장 세팅 매뉴얼
📡 네트워크 구성
갤럭시북 (Qualcomm 제공 노트북)
→ 주 네트워크 호스트 역할 수행
→ 학교 와이파이 연결 후 핫스팟 개설

핫스팟에 연결되는 장비 총 3대

라즈베리파이 (Raspberry Pi)

RubikPi (QCS6490)

여분 노트북 (Arduino 전원 및 제어 전용)

→ 총 4개 호스트가 로컬 네트워크 구성

⚙️ 하드웨어 세팅 순서
1. 갤럭시북 부팅 및 네트워크 준비
갤럭시북을 부팅 후 학교 와이파이에 연결

핫스팟을 개설하여 다른 장비들이 접속할 수 있도록 준비

2. 키오스크 메인 전원 인가
키오스크 박스 내부 멀티탭의 전원만 인가

연결된 구성:

라즈베리파이 어댑터

터치 디스플레이 어댑터

주의: L298N 외부 전원 플러그는 아직 연결하면 안됨!

3. RubikPi 연결 및 테스트
RubikPi에 카메라 연결

여분 멀티탭에 RubikPi 어댑터 연결 → 전원 인가

RubikPi의 카메라가 의자를 향하도록 위치 조정

test.py 실행하여 탐지 테스트 수행

4. Arduino 연결
여분 노트북에 Arduino 연결 (USB)

아두이노 전원이 USB를 통해 공급됨

5. L298N 외부 전원 인가
가장 마지막 단계로 L298N의 외부 전원 플러그를 멀티탭에 연결
→ 모터 드라이버 작동 준비 완료

6. 메인 프로그램 실행
RubikPi: main.py 실행

여분 노트북: 아두이노 제어용 main.py 실행

7. 라즈베리파이 키오스크 앱 실행
라즈베리파이에서 크로미움 브라우저로 키오스크 앱 실행

이로써 전체 시스템 준비 완료

⚠️ 주의사항
L298N 외부 전원은 반드시 마지막에 인가

카메라 각도 조정은 테스트로 정확히 확인

전원 연결은 항상 순서대로 진행
