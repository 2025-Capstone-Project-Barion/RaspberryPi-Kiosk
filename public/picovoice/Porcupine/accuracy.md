# Picovoice Porcupine 학습 설정

## ✅ 최종 선택: `"Hey Barry on"` (언어: English)
![image](https://github.com/user-attachments/assets/5324b2b3-df52-4949-b3a8-4530e5c6a45a)


- **선택 언어**는 `English`로 한다.
- **학습 문장**은 `"Hey Barry on"`으로 설정한다.

### 💡 선택 이유

- `"Barry"`의 발음은 /ˈbæri/로 `"배리"`와 거의 동일하다.
- `"on"`은 /ɑn/으로 `"온"`과 유사하여 `"배리온"`과 자연스럽게 이어진다.
- `"Hey Barry on"`의 전체 발음은 /heɪ ˈbæri ɑn/으로 `"헤이 배리온"`과 약 90% 이상 유사하다.
- **Picovoice 콘솔 테스트 결과**, 다양한 억양과 속도에서도 `"Hey Barry on"`이 **가장 높은 인식률**을 보였다.
- 한국어 억양 사용자도 `"헤이 배리온"`처럼 발음하면 정확하게 인식된다.
- `"헤이 배리온"`이라는 한글 발음은 실제 사용 의도와 브랜드명 `"Barion"`과도 직관적으로 잘 연결된다.

---

## 🟦 대체안: `"헤이 배리온"` (언어: Korean)

- **선택 언어**는 `Korean`으로 한다.
- **학습 문장**은 `"헤이 배리온"`으로 설정할 수 있다.

### ⚠️ 한계

- `"헤이 배리온"`은 Picovoice Korean 모델에서도 인식 가능하지만,
- 영어 모델 `"Hey Barry on"`보다 **전반적인 인식률은 낮은 편**이다.
- 한국어 발음 억양이 일정하지 않으면 `"해이"`, `"베리온"` 등으로 오인될 수 있다.

---

## 🔍 테스트한 후보들 (정리)

### Korean 기준

| 문장        | 결과 | 비고                                              |
| ----------- | ---- | ------------------------------------------------- |
| 헤이 배리온 | ⚠    | 인식 가능하나, 영어식보다 약간 낮은 정확도        |
| 헤이 베리온 | ⛔   | `"베"` 발음이 `"Berry"`로 처리될 가능성           |
| 해이 배리온 | ⛔   | `"해이"`가 부자연스러움. Picovoice 처리 실패 우려 |

### English 기준

| 문장             | 결과 | 비고                                                           |
| ---------------- | ---- | -------------------------------------------------------------- |
| **Hey Barry on** | ✅   | 영어식/한국어식 발음 모두 대응 가능. 가장 안정적인 인식률 제공 |
| Hey Berry on     | ⚠    | `"Berry"`는 유사하지만 오탐 우려 있음                          |
| Hey Bari on      | ⛔   | r 발음 약하면 실패 가능성 높음                                 |
| Hey Barion       | ⛔   | 고유어라 G2P 처리 불안정                                       |

## ✅ 최종 정리

| 언어    | 추천 문장          | 적합도     | 코멘트                                                |
| ------- | ------------------ | ---------- | ----------------------------------------------------- |
| English | **"Hey Barry on"** | ⭐⭐⭐⭐⭐ | 테스트 결과 최고 인식률. 한국어 발음에도 매우 잘 대응 |
| Korean  | **"헤이 배리온"**  | ⭐⭐⭐⭐   | 실제 발음과 유사하지만, 인식률은 영어보다 살짝 낮음   |

> 💬 현재 기준으로는 **"Hey Barry on" (English)** 설정이  
> 가장 안정적이며, `"헤이 배리온"` 발음 그대로 사용해도  
> 높은 인식률을 보장하므로 **최종 채택 버전으로 적합하다.**
