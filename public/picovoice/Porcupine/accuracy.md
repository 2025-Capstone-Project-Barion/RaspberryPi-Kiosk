# Picovoice Porcupine 학습 설정

## ✅ 최종 선택: "헤이 배리온" (언어: Korean)

- **선택 언어**는 `Korean`으로 한다.
- **학습 문장**은 `"헤이 배리온"`으로 설정한다.

### 💡 선택 이유

- 키오스크 사용자 대상이 한국인이므로 실제 호출 발음은 `"헤이 배리온"`이 가장 자연스럽다.
- Picovoice는 발음을 기준으로 wake word를 인식하므로, 한국어 억양에 맞게 학습하는 것이 인식률을 높이는 최선이다.
- `"헤이 배리온"`은 충분한 음소 수를 가지며, 오탐을 방지하는 데 유리하다.
- 브랜드명 `"Barion"`과도 직관적으로 연관된다.

---

## ⚠️ 발음 유사 표현 주의사항: `"헤이"` vs `"해이"`

- `"헤이"`는 영어 `"Hey"`의 발음 /heɪ/와 유사하여 **wake word로 안정적으로 인식**된다.
- 반면 `"해이"`는 /hɛ.i/처럼 두 음절로 처리되어 Picovoice에서 **다른 단어로 오인되거나 인식이 되지 않을 수 있다.**
- 따라서, 사용자에게 **“헤이 배리온”**처럼 발음하도록 유도하는 것이 바람직하다.

---

## 🔍 테스트한 후보들 (Korean 기준)

| 문장            | 결과 | 이유                                                  |
| --------------- | ---- | ----------------------------------------------------- |
| **헤이 배리온** | ✅   | 실제 발음과 가장 유사하며, 인식률이 높다.             |
| 헤이 베리온     | ⛔   | `"베"`가 `"Berry"`로 인식될 가능성이 있어 부적절하다. |
| 헤이 바리온     | ⛔   | `"바"` 발음이 어색하고 정확도가 낮다.                 |
| 해이 베리온     | ⛔   | `"해이"`는 억양이 부자연스럽고 인식률이 낮다.         |

## 🟦 대체안: "Hey Barry on" (언어: English)

- **선택 언어**는 `English`로 한다.
- **학습 문장**은 `"Hey Barry on"`으로 설정한다.

### ✅ 선택 이유

- `"Barry"`의 발음은 /ˈbæri/로 `"배리"`와 거의 동일하다.
- `"on"`은 /ɑn/으로 `"온"`과 유사하여 `"배리온"`과 자연스럽게 이어진다.
- `"Hey Barry on"`의 전체 발음은 /heɪ ˈbæri ɑn/으로 `"헤이 배리온"`과 약 90% 유사하다.
- Picovoice 콘솔 테스트 결과, 다양한 억양과 속도에서도 `"Hey Barry on"`이 **가장 높은 인식률**을 보였다.
- 사용자 중 일부는 영어식 억양으로 자연스럽게 호출할 가능성도 있으므로, 영어 발음 기반의 대안으로 적절하다.

## 📝 최종 정리

| 언어    | 추천 문장          | 적합도     | 코멘트                                                    |
| ------- | ------------------ | ---------- | --------------------------------------------------------- |
| English | **"Hey Barry on"** | ⭐⭐⭐⭐⭐ | Picovoice 테스트에서 가장 안정적. 한국어 억양도 잘 대응함 |
| Korean  | **"헤이 배리온"**  | ⭐⭐⭐⭐   | 실제 발음과 일치하지만, English 모델보다 인식률은 낮음    |

- 언어 설정은 **사용자 억양과 실제 발음**을 고려해 결정해야 한다.
- 현재 기준으로는 **"Hey Barry on" (English)** 이 가장 안정적이며 실제 테스트에서도 우수한 성능을 보인다.
- `"헤이"`처럼 발음되도록 안내하고, `"해이"`와 같은 변형 발음은 피하는 것이 좋다.
